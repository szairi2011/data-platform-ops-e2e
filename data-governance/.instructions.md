## OpenMetadata

- Self-contained stack (own MySQL + Elasticsearch) — does not share DBs with other stacks
- Accessible at `http://localhost:8585` (default admin/admin)
- Hive Metastore connector configured to ingest table schemas from `yarn-cluster` HMS
- OpenLineage events arrive at `http://openmetadata:4343/api/v1/lineage` (OpenLineage endpoint)

---

## OpenLineage Integration

All Spark jobs (`BronzeIngester`, `SilverTransformer`, `GoldAggregator`) must set:

```bash
--conf spark.extraJavaOptions=-javaagent:/opt/openlineage/openlineage-spark-*.jar
--conf spark.openlineage.transport.type=http
--conf spark.openlineage.transport.url=http://openmetadata:4343
--conf spark.openlineage.namespace=data-platform
```

`openlineage.env` exports these values; Airflow `SparkSubmitOperator` reads them via `--conf` entries.

## Great Expectations

### Silver Suite (`silver_suite.json`)
- `expect_column_to_exist`: `event_date`, `entity_id`, `metric_value`
- `expect_column_values_to_not_be_null`: `event_date`, `entity_id`
- `expect_column_values_to_be_between`: `metric_value` min=0, max=1000000
- `expect_table_row_count_to_be_between`: min=1

### Gold Suite (`gold_suite.json`)
- `expect_column_to_exist`: `event_date`, `entity_id`, `daily_total`
- `expect_column_values_to_not_be_null`: all columns
- `expect_column_values_to_be_between`: `daily_total` min=0
- `expect_column_pair_values_A_to_be_greater_than_B`: `daily_total` > 0 when `record_count` > 0

### Checkpoints
- `silver_checkpoint.yml` — reads Silver Parquet from HDFS, runs `silver_suite`
- `gold_checkpoint.yml` — reads Gold Parquet from HDFS, runs `gold_suite`
- Both run as `BashOperator` tasks in Airflow `medallion_pipeline` DAG

## SDD Rules

1. OpenMetadata stack must be self-contained — no shared MySQL or Elasticsearch with other services
2. OpenLineage transport URL must be configurable via env var (not hardcoded in Spark job code)
3. GE checkpoints run after the corresponding Spark job succeeds (Airflow task dependency)
4. GE validation failures must set Airflow task state to FAILED (non-zero exit from GE CLI)
5. Do not add Marquez — OpenMetadata is the single lineage + catalog backend
